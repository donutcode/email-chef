# email-chef

EmailChef takes HTML and CSS and compiles them into inline `.html` files

## installation

    $ git clone git://github.com/donutcode/email-chef.git
    $ npm install -g email-chef

## update

    $ cd email-chef
    $ git pull origin master
    $ npm install -g .
